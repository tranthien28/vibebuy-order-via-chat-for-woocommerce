VibeBuy Lite is a modern, high-conversion "Chat-to-Buy" WordPress plugin designed specifically for WooCommerce.

== Description ==

VibeBuy Lite empowers customers to skip the traditional cart checkout flow and order products directly via popular messaging platforms like WhatsApp and Telegram.

Instead of utilizing legacy WordPress UI blocks, VibeBuy sets a new standard by seamlessly integrating a React-powered SPA right within the WordPress Admin dashboard, providing store owners with a breathtaking, app-like configuration experience.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/vibebuy-order-connect-lite` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the VibeBuy Lite menu to configure your messaging handles.
